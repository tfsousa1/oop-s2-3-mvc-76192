# Acme Global College (VGC)

ASP.NET Core MVC application for a multi-branch student and course management system.

## Tech Stack
- ASP.NET Core MVC
- Entity Framework Core
- ASP.NET Core Identity
- SQL Server
- xUnit
- GitHub Actions
- Serilog

## Project Structure
- `src/AcmeGlobalCollege.Web` - main MVC application
- `tests/AcmeGlobalCollege.Tests` - unit tests

## Current Progress
- Solution structure created
- MVC web app created
- Test project created
- EF Core and Identity configured
- GitHub Actions CI added
